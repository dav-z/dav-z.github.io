ART.work

Github | Live Demo

ART.work is a collaborative and interactive web app made for artists. As a user, you can upload some of your previous art pieces into your portfolio, update your friends and followers on your recent projects, and share some freelance work that you think might interest other artists.

Some technologies I used are Ruby on Rails, JavaScript, jQuery, Bootstrap, HTML, and CSS. Check out the Github to see more information.

Urban Gardener

Github | Live Demo

Urban Gardener is an app for plant lovers in urban areas. It is made for people who love to plant but are limited to space. As a user, you can add plants to your account and view their care information, make posts about your planting progress, and use our search feature to search for recipes using your edible plants!

This project used Ruby on Rails, the Food2Fork API, JavaScript, jQuery, Bootstrap, HTML, and CSS, Check out the Github to see more information.

JavaScript Jukebox

Github | Live Demo

The JavaScript Jukebox was one of my first projects using Object-Oriented JavaScript! It features a hidable interactive playlist, which allows you to play any song on it with a click! It also features a play/pause function, a random song function, as well as skip and rewind functions as well.

Test out this jukebox and listen to some pretty amazing songs!
